🌤️ Weather App

A modern, responsive Weather Application that fetches real-time weather data using a public API. Built with simplicity and performance in mind, this app allows users to quickly check weather conditions for any city worldwide.

✨ Features

- 🔍 Search weather by city name
- 🌡️ Displays temperature, weather condition, and humidity
- ⚡ Real-time data fetching from API
- 🎨 Clean, responsive, and user-friendly UI
- 📱 Works across desktop and mobile devices

---

🛠️ Technologies Used

- HTML5 – Structure
- CSS3 – Styling & responsiveness
- JavaScript (ES6) – Logic & API handling
- OpenWeather API – Weather data source

---

⚙️ How It Works

1. Enter a city name in the search bar
2. The app sends a request to the weather API
3. Receives real-time weather data
4. Displays the results instantly on the UI
