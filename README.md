# 🌤️ Weather App

A simple and responsive Weather Application that fetches real-time weather data using a public API.

## 🚀 Features
- Search weather by city name
- Displays temperature, weather condition, and humidity
- Dynamic API data fetching
- Clean and user-friendly UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## 📌 How It Works
Enter a city name, and the app fetches live weather data from the API and displays it instantly on the webpage.

## 💻 Project Setup
1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter your API key in the JavaScript file
